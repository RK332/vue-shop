# vue-shop

一个基于 Vue 2 和 Element UI 实现的后台电商管理系统学习项目，用来练习 Vue 全家桶、后台权限管理、数据可视化等常见功能。

## 功能简介

- **用户管理**：用户列表、添加 / 编辑 / 删除、角色分配  
- **权限管理**：角色管理、权限分配、菜单 / 按钮级别权限控制  
- **商品管理**：商品分类、商品参数、商品列表、添加商品  
- **订单管理**：订单列表、订单详情、物流信息（示例）  
- **数据统计**：使用 ECharts 展示图表报表  
- **富文本编辑**：基于 `vue-quill-editor` 的富文本编辑器  
- **加载进度条**：使用 `nprogress` 展示接口加载进度  
- **表格树形控件**：使用 `vue-table-with-tree-grid` 展示树形表格  

> 说明：本项目主要用于前端练习和学习，不适合作为生产环境项目直接使用。

## 技术栈

- **前端框架**：Vue 2.x  
- **路由管理**：Vue Router  
- **UI 组件库**：Element UI  
- **可视化**：ECharts  
- **富文本编辑**：vue-quill-editor  
- **HTTP 请求**：Axios  
- **构建工具**：Vue CLI 4
